## Eclipse Syndrome

### 1. 프로젝트 개요
* 언리얼 엔진5를 사용하여 만든 아포칼립스 배경의 슈팅 게임
  * C++ (95%)
  * BP (애니메이션 및 테스트)
* 25.2.12 ~ 25.3.7 동안 개발
* 5명의 팀원
  * 담당 분야 : 캐릭터, 무기, 인벤토리 및 UI, 게임 시스템
  
### 2. 개발 환경
* Unreal Engine 5.5
* Visual Studio 2022
* Git Kraken

### 3. 프로젝트 담당 부분
* Character 폴더
  * PlayerCamera
  * PlayerCharacter : Grapple 및 Drone 빙의 기능을 제외한 나머지 부분
  * PlayerCharacterController : Grapple 및 Drone 빙의 기능을 제외한 나머지 부분
  * PlayerReloadAnimNotify
  * PlayerRollAnimNotify
* System 폴더
  * DefaultGameInstance
  * DefaultGameMode
  * DefaultGameState
* Volume 폴더
  * LevelChangeActor
* Weapon 폴더
  * Weapon
  * NewWeapon
  * WeaponAR1
  * WeaponAR2
  * WeaponSR
  * WeaponShell
  * WeaponShotgun
